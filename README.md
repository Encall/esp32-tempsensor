# ESP32 Arduino Temperature Sensor Station
Arduino ESP32 based temperature sensor with web-interface, data collection and line notify.
  Component
  *   DHT22
  *   Wemos D1 R32
  *   LCD 16x2 I2c
  *   HLK 5M05
  
  Web interface require database.
